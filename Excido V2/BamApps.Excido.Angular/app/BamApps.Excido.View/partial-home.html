Excido is a sample, full-stack, Single Page Application made by Bill Fugina using Angular and the Microsoft .NET framework.

If you would like to follow the progress or learn more about the tools and technologies used in building this app, please visit my other site, BAM Apps.

The plan is for Excido to become a content sharing service where content can be set to expire and disappear. At the moment, it is simply a short-link service. You may create a short link in the {{preLogin.NameSpace}} namespace that redirects to any other valid link.

Paste a long URL and an expiration date and Excido will generate a short link for you.
